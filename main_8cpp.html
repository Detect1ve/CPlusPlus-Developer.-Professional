Detailed Description

Demonstration of different ways to print IP addresses using templates.

Author
Nikolay Grin
Date
2025

The program demonstrates the use of C++ templates to print IP addresses of various data types: integers, strings, containers, and tuples. For each data type, a specialized version of the print_ip function is used.

template<typename T>
std::enable_if_t< std::is_integral_v< T > > print_ip ( T const & ip_address)

Printing IP address for integer types.

Converts an integer value to IP address format, separating bytes with dots. For example, int32_t(2130706433) will be output as "127.0.0.1". The definition of the integral type is done via std::is_integral_v.

Output IP address for tuples.

Prints an IP address represented as a tuple, where all elements have the same type. For example, std::tuple(127, 0, 0, 1) will be printed as "127.0.0.1".
